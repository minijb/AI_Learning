# 学习计划: Mermaid 语法

> 创建日期: 2026-06-11
> 预计总耗时: ~6 小时
> 目标水平: 熟练（能独立绘制项目所需各类图表）

---

## 学习目标

完成本计划后，你将能够：

- 在 Markdown 文档中嵌入 Mermaid 图表，覆盖流程图、时序图、类图、状态图、ER 图、甘特图、Git 图、思维导图、饼图等九种常用图表类型
- 使用 Obsidian、GitHub、GitLab、Notion 等主流平台的原生 Mermaid 渲染
- 通过自定义样式和主题让图表符合团队或项目视觉规范
- 为技术文档、设计文档、项目计划、架构评审编写清晰的可维护图表
- 理解 Mermaid DSL 的设计约束与常见陷阱，写出能正确渲染的图表

## 前置要求

- [ ] 熟悉 Markdown 基本语法（代码块、标题、列表）
- [ ] 有 Obsidian 或任意支持 Markdown 的编辑器可用
- [ ] 无需编程基础，无需安装额外软件
