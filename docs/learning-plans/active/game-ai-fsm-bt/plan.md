# 学习计划: 状态机与行为树 — 游戏 AI 开发

> 创建日期: 2026-06-02
> 预计总耗时: 40-50 小时（20 节 × 2-2.5 小时/节）
> 目标水平: 资深游戏开发工程师（面试/实战级）

---

## 学习目标

完成本计划后，你将能够：

- 从零实现多种状态机架构（简单 FSM、分层 HSM、Pushdown Automata），并在 Unity/Unreal/Lua 三种环境中运行
- 从零实现完整的行为树引擎，包括 Composite、Decorator、Leaf、Service、Parallel 全部节点类型
- 深度理解 Unreal Engine 原生 Behavior Tree 系统的源码级设计（UBehaviorTree、UBTNode、BTTask、BTDecorator、BTService、Blackboard）
- 设计可扩展、高性能的游戏 AI 架构，在 100+ 实体场景下保持 60fps
- 在系统设计面试中建模复杂游戏 AI 问题（敌人 AI、NPC 日常、群组战术）
- 对比评估 FSM / HSM / BT / Utility AI / GOAP / HTN 的技术选型，给出决策理由

---

## 前置要求

- [x] C++ 基础：类、继承、虚函数、模板基础（参考 [[../../../knowledge-notes/cpp-special-member-rules|cpp-special-member-functions]]）
- [x] C# 基础：类、接口、泛型、委托/事件
- [x] Lua 基础：table、metatable、闭包、协程
- [x] Unity 基础：MonoBehaviour、GameObject、Update/LateUpdate
- [x] Unreal Engine 基础：AActor、UActorComponent、Tick、UObject 系统

> 如果不满足以上前置要求，建议先补充基础，再回到本计划。
