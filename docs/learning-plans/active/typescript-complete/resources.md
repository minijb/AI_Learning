# 学习资源: TypeScript 完整学习

## 官方文档

- [TypeScript 官方文档](https://www.typescriptlang.org/docs/) — 最权威的语言参考。
- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html) — 从基础到高级类型的系统教程。
- [tsconfig 参考](https://www.typescriptlang.org/tsconfig) — 每个配置项的详细说明和示例。
- [Robin Wieruch: TypeScript in a Node.js Project](https://www.robinwieruch.de/typescript-node-js/) — 2025 年实操指南，涵盖 `tsx`、ESM、环境变量配置。
- [TypeScript 5.8 发布说明](https://devblogs.microsoft.com/typescript/announcing-typescript-5-8/) — `--module nodenext` 支持 `require()` ESM、`--erasableSyntaxOnly`、返回表达式分支细粒度检查等。
- [DefinitelyTyped 贡献指南](https://definitelytyped.org/guides/contributing.html) — 学习如何编写和维护 `.d.ts` 声明文件。

## 推荐书籍

- *Programming TypeScript* — Boris Cherny 著，适合有编程背景的开发者系统学习。
- *Effective TypeScript* — Dan Vanderkam 著，聚焦 62 条实用建议，适合进阶。
- *TypeScript Cookbook* — 按问题组织的实战配方，适合作为案头参考。

## 在线教程

- [Total TypeScript](https://www.totaltypescript.com/) — Matt Pocock 的免费与付费课程，类型体操讲解极细。
- [TypeScript Deep Dive](https://basarat.gitbook.io/typescript/) — 开源电子书，覆盖语言深层细节。
- [Execute Program: TypeScript](https://www.executeprogram.com/courses/typescript) — 间隔重复式练习，适合巩固。
- [type-challenges/type-challenges](https://github.com/type-challenges/type-challenges) — 类型体操题库，适合练习 `Pick`、`Omit`、`infer`、模板字面量类型等。
- [@tsconfig/bases](https://github.com/tsconfig/bases) — 社区维护的 tsconfig 预设，如 `@tsconfig/node-lts`。

## 社区资源

- [TypeScript Weekly](https://typescript-weekly.com/) — 每周精选文章和库。
- [r/typescript](https://www.reddit.com/r/typescript/) — 讨论区，可跟踪常见问题和最佳实践。
- [TypeScript GitHub Discussions](https://github.com/microsoft/TypeScript/discussions) — 官方讨论区，适合提设计问题。

## 开源项目

- [zx](https://github.com/google/zx) — Google 的工具脚本库，学习如何用 TS 写命令行脚本。
- [commander.js](https://github.com/tj/commander.js) — 流行的 Node.js CLI 参数解析库，带 TypeScript 类型。
- [tsup](https://github.com/egoist/tsup) — 基于 esbuild 的 TS 打包工具，适合构建库。
- [tsx](https://github.com/privatenumber/tsx) — TS 执行环境，本计划部分示例会使用它运行代码。
