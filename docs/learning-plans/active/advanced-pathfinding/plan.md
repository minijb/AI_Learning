# 学习计划: 高阶寻路系统 (Advanced Pathfinding Systems)

> 创建日期: 2026-05-31
> 预计总耗时: 约 30 小时（30 节 × ~45-60 分钟）
> 目标水平: 精通
> 技术栈: C++ 核心算法 + Unity C# 可视化 + Unity ECS 集成

---

## 学习目标

完成本计划后，你能：

1. **手写核心算法** — 从零实现 A*、JPS、Theta*、D* Lite、Flow Field、ORCA，理解每一步的数学推导
2. **使用工业库** — 集成 Recast/Detour，理解其体素化→区域生长→轮廓→三角剖分的完整管线
3. **设计寻路架构** — 搭建生产级网格系统，含地形感知、岛屿检测、模糊惩罚、KD-Tree 加速
4. **处理多智能体** — 实现 Flow Field 批量寻路 + ORCA 局部避障，支持数百单位同时移动
5. **集成 Unity ECS** — 将寻路系统接入 Burst/Jobs/ECS，达到生产级性能
6. **烘焙管线** — 理解 Authoring→Baking 的离线数据生成流程
