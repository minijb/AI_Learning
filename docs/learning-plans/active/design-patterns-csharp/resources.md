# 学习资源: 设计模式 (C# + C++)

## 官方文档

- [.NET Guide — Microsoft Learn](https://learn.microsoft.com/en-us/dotnet/) — C# 语言特性和 .NET API 官方参考
- [C# Programming Guide](https://learn.microsoft.com/en-us/dotnet/csharp/programming-guide/) — 泛型、委托、事件等核心特性
- [cppreference.com](https://en.cppreference.com/) — C++ 标准库和语言特性的权威参考

## 推荐书籍

- **《Design Patterns: Elements of Reusable Object-Oriented Software》** — GoF 四人组经典，所有模式的原始定义
- **《Head First Design Patterns》** — 图文并茂的入门书，适合第一遍通读
- **《C# Design Patterns: A Guide to Building maintainable and scalable Software》** — Esther Vasquez、Oscar R. Paredes — C# 原生示例
- **《Design Patterns in C#: A Hands-on Guide with Real-world Examples》** — Vaskaran Sarcar — 含完整可运行代码
- **《Refactoring: Improving the Design of Existing Code》** — Martin Fowler — 重构与模式的关系
- **《Modern C++ Design》** — Andrei Alexandrescu — C++ 模板元编程和 Policy-based design 的经典之作
- **《C++ Software Design: Design Principles and Patterns for High-Quality Software》** — Klaus Iglberger — 现代 C++ 设计模式，含值语义、模板等
