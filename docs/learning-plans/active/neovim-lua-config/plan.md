# 学习计划: Neovim + Lua 配置实战

> 创建日期: 2026-06-01
> 预计总耗时: 约 16 小时
> 目标水平: 进阶 — 能独立用 Lua 配置和定制 Neovim

---

## 学习目标

完成本计划后，你将能够：

- 用 Lua 编写结构清晰的 Neovim 配置文件
- 使用 lazy.nvim 管理插件，理解懒加载和事件驱动加载
- 独立配置 LSP（代码补全、跳转、诊断）、Treesitter（语法高亮）、nvim-cmp（补全引擎）
- 自定义按键映射、自动命令、用户命令
- 搭建一套外观统一、功能齐全的开发环境
- 遇到配置问题时能定位到 Neovim 的对应 Lua API 文档

## 前置要求

- [x] 会使用 Neovim 的基本编辑操作（hjkl、模式切换、保存退出）
- [x] 有任意一门编程语言的基础（变量、函数、控制流）
- [ ] Neovim 0.9+ 已安装（计划覆盖 API 到 0.10 稳定版）
