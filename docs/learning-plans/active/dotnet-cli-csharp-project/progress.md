# dotnet CLI 与 C# 工程构建 — 学习进度

## 第一阶段：环境与基础命令

- [ ] 01. dotnet 环境安装与验证 (40min)
- [ ] 02. `dotnet new` — 创建项目与解决方案 (60min)
- [ ] 03. `dotnet build` / `run` / `clean` — 编译运行与清理 (60min)

## 第二阶段：依赖与解决方案管理

- [ ] 04. `dotnet add` / `remove` — 管理包引用与项目引用 (60min)
- [ ] 05. `dotnet sln` — 解决方案管理 (50min)
- [ ] 06. 项目文件结构深入 — `.csproj` 与 `.sln` (60min)

## 第三阶段：测试、发布与打包

- [ ] 07. `dotnet test` — 单元测试 (60min)
- [ ] 08. `dotnet publish` / `pack` — 发布与打包 (60min)
- [ ] 09. 多目标框架与条件编译 (50min)

## 第四阶段：扩展工具与实战

- [ ] 10. `dotnet tool` — 全局与本地工具 (45min)
- [ ] 11. `dotnet ef` — EF Core CLI 工具 (60min)
- [ ] 12. 综合实战 — 构建完整多层项目 (90min)

## 里程碑

- [ ] 第一阶段完成：能用 dotnet CLI 创建、编译、运行一个控制台项目
- [ ] 第二阶段完成：能管理多项目解决方案、理解 `.csproj` 和 `.sln` 的结构
- [ ] 第三阶段完成：能编写单元测试、发布应用、打包类库
- [ ] 最终目标：从零搭建一个完整的多层 C# 解决方案
