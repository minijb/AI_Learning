# 推荐学习资源汇总

> 游戏性能优化全攻略 — 按类别整理

---

## 必读网站/文档

- [GPUOpen Performance](https://gpuopen.com/performance/) — AMD GPU 性能优化官方指南
- [NVIDIA Developer — GameWorks & Optimization](https://developer.nvidia.com/gameworks) — NVIDIA 性能优化资源
- [RenderDoc 官方文档](https://renderdoc.org/docs/) — 最强免费 GPU 调试器
- [Tracy Profiler](https://github.com/wolfpld/tracy) — C++ 高性能 CPU/GPU Profiler
- [UE5 Performance Guidelines](https://docs.unrealengine.com/5.0/en-US/performance-guidelines-for-artists-and-designers-in-unreal-engine/) — UE 官方性能指南
- [Unity Performance Optimization](https://docs.unity3d.com/Manual/Performance.html) — Unity 官方性能文档
- [Real-Time Rendering Resources](https://www.realtimerendering.com/) — 实时渲染权威资源聚合

## 经典书籍

| 书名 | 作者 | 覆盖章节 | 难度 |
|------|------|---------|------|
| *Real-Time Rendering* (4th) | Akenine-Möller et al. | 全部渲染章节 | 进阶 |
| *Game Engine Architecture* (3rd) | Jason Gregory | CPU/内存/多线程 | 中阶 |
| *GPU Gems* 1-3 (免费在线) | NVIDIA | GPU/Shader 优化 | 进阶 |
| *Data-Oriented Design* (免费在线) | Richard Fabian | DOD/ECS | 中阶 |
| *Video Game Optimization* | Eric Preisz / Ben Garney | 全局优化方法 | 入门 |
| *Unity 实战性能优化* | Unity Technologies | Unity 全部 | 入门 |

## 在线课程/视频

- [GDC Vault — Programming Track](https://www.gdcvault.com/) — GDC 历年性能优化演讲（部分免费）
- [Handmade Hero](https://handmadehero.org/) — Casey Muratori 从零构建游戏引擎（C 语言，性能优先）
- [Computer Graphics — CMU 15-462/662](http://15462.courses.cs.cmu.edu/) — 卡内基梅隆大学图形学课程
- [CppCon — Data-Oriented Design](https://www.youtube.com/watch?v=rX0ItVEVjHc) — Mike Acton 经典 DOD 演讲

## 开源项目学习

- [Filament](https://github.com/google/filament) — Google 的 PBR 渲染引擎（Android/跨平台）
- [The-Forge](https://github.com/ConfettiFX/The-Forge) — 跨平台渲染框架（学习 GPU API 优化）
- [BGFX](https://github.com/bkaradzic/bgfx) — 跨平台渲染库（Draw Call/批处理参考）
- [EnTT](https://github.com/skypjack/entt) — C++ ECS 库（学习 ECS 实现）
- [UE4 Gameplay Ability System Sample](https://github.com/tranek/GASDocumentation) — UE GAS 性能参考
