# 学习资源: Git 进阶——从日常使用到底层原理

## 官方文档

- [Pro Git Book（中文版）](https://git-scm.com/book/zh/v2) — 官方权威书籍，免费在线，第 7/10 章讲工具与 internals
- [Git 官方参考手册](https://git-scm.com/docs) — 所有命令的权威文档
- [Git 官方教程](https://git-scm.com/doc) — 入门到进阶官方路径

## 进阶书籍

- **《Pro Git》** Scott Chacon & Ben Straub — 最权威的 Git 书，第 10 章 "Git Internals" 是底层原理必读
- **《Version Control with Git》** Jon Loeliger & Matthew McCullough — 偏重 internals 和高级用法

## 底层原理（Internals）

- [Git Internals Chapter (Pro Git)](https://git-scm.com/book/zh/v2/Git-Internals-Plumbing-and-Porcelain) — Plumbing vs Porcelain、对象模型、refspec
- [Git from the Bottom Up](https://jwiegley.github.io/git-from-the-bottom-up/) — 从底层对象开始自下而上讲解
- [Write yourself a Git!](https://wyag.thb.lt/) — 用 Python 手写一个迷你 Git，深刻理解对象模型
- [Git Internals: How Git Really Works](https://www.youngju.dev/blog/culture/2026-03-24-git-internals-how-git-works-deep-dive.en) — Blob/Tree/Commit/DAG 深度剖析

## 高级用法与工作流

- [Advanced Git Techniques (sanj.dev)](https://sanj.dev/post/advanced-git-techniques-for-modern-development) — rebase、reflog、cherry-pick 实战
- [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials) — 交互式图解，merge vs rebase 讲得最清楚
- [Learn Git Branching](https://learngitbranching.js.org/?locale=zh_CN) — 可视化交互式练习，强烈推荐配合本计划使用
- [First Aid Git](https://firstaidgit.io/) — 常见 Git 灾难的可搜索救援指南

## Hooks 与自动化

- [pre-commit framework](https://pre-commit.com/) — 多语言 hooks 管理框架，业界标准
- [Husky](https://typicode.github.io/husky/) — Node.js 项目的 Git hooks 工具
- [lefthook](https://github.com/evilmartians/lefthook) — 快速、并发的 Git hooks 管理器

## 可视化工具

- [GitKraken](https://www.gitkraken.com/) — 跨平台图形化 Git 客户端，历史可视化优秀
- [lazygit](https://github.com/jesseduffield/lazygit) — 终端 TUI 客户端，rebase/commit 交互体验极佳
- [tig](https://jonas.github.io/tig/) — 终端历史浏览器

## 社区资源

- [Git 官方邮件列表与贡献指南](https://git-scm.com/community)
- [Stack Overflow `git` 标签](https://stackoverflow.com/questions/tagged/git) — 大量真实事故案例
