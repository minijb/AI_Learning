# 学习计划: Oh My Pi Code Agent 使用指南

> 创建日期: 2026-05-26
> 预计总耗时: 约 12 小时（12 节 × 约 60 分钟）
> 目标水平: 入门到进阶

---

## 学习目标

完成本计划后，你将能够:

- 独立使用 OMP 进行日常编码工作：阅读代码、搜索代码库、精确编辑
- 利用 LSP 进行代码智能操作（跳转定义、查找引用、重命名）
- 使用子代理（subagent）并行处理多个任务
- 配置和定制 OMP：设置、技能（Skills）、自定义工具、扩展
- 集成外部服务（MCP 服务器）
- 运用生产级最佳实践：安全、会话管理、有效提示

## 前置要求

- [ ] 熟悉命令行操作（终端、Shell）
- [ ] 有 Git 使用经验
- [ ] 有任意编程语言的项目经验
- [ ] 已安装 Bun 运行时（OMP 基于 Bun）

## 里程碑

- [ ] 第一阶段 — 基础操作: 完成第 1-4 节，能用 OMP 阅读和编辑代码
- [ ] 第二阶段 — 高效工作: 完成第 5-7 节，能用 LSP 和子代理加速开发
- [ ] 第三阶段 — 定制化: 完成第 8-10 节，能配置 OMP 并编写 Skill/工具
- [ ] 第四阶段 — 集成与实战: 完成第 11-12 节，能用 MCP、掌握生产实践
- [ ] 最终项目: 为你的实际项目配置完整的 OMP 工作环境（含 AGENTS.md、Skills、MCP）
