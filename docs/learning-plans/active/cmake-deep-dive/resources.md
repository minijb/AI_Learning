# CMake 深度学习 — 推荐资源

## 官方文档

- [CMake 官方文档 (4.3.3)](https://cmake.org/cmake/help/latest/) — 最权威的参考，搜索功能强大，可切换版本
- [CMake Tutorial (官方教程)](https://cmake.org/cmake/help/latest/guide/tutorial/index.html) — 10 步渐进式教程，覆盖基础到安装
- [cmake-generator-expressions(7)](https://cmake.org/cmake/help/latest/manual/cmake-generator-expressions.7.html) — 生成器表达式完整参考
- [cmake-presets(7)](https://cmake.org/cmake/help/latest/manual/cmake-presets.7.html) — Presets JSON 格式规范
- [cmake-toolchains(7)](https://cmake.org/cmake/help/latest/manual/cmake-toolchains.7.html) — 工具链文件与交叉编译指南
- [FetchContent 模块文档](https://cmake.org/cmake/help/latest/module/FetchContent.html) — FetchContent API 完整参考
- [install() 命令文档](https://cmake.org/cmake/help/latest/command/install.html) — 安装规则参考
- [CTest 手册](https://cmake.org/cmake/help/latest/manual/ctest.1.html) — CTest 命令行参考

## 书籍

- **Professional CMake: A Practical Guide** (Craig Scott) — 最全面的 CMake 书籍，持续更新到 CMake 4.x
- **Modern CMake for C++** (Rafael Gago) — 面向 C++ 开发者的现代 CMake 指南
- **CMake Best Practices** (Dominik Berner, Mustafa Kemal Gilor) — 实践导向的最佳实践集合

## 在线教程与指南

- [An Introduction to Modern CMake (Henry Schreiner)](https://cliutils.gitlab.io/modern-cmake/) — 社区公认的 Modern CMake 入门书，在线免费
- [Effective Modern CMake (Gist)](https://gist.github.com/mbinna/c2789b3d81e6e2e6e780) — 精炼的 Do's and Don'ts 清单
- [HSF CMake Training](https://hsf-training.github.io/hsf-training-cmake-webpage/) — HEP Software Foundation 培训材料

## 演讲视频

- **Starting my modern C++ Project with CMake in 2024** — Jens Kreitlow (Meeting C++ 2024) — 从零搭建现代 C++ 项目
- **Post-Modern CMake - From 3.0 to 4.0** — Vito Gamberini (CppNow 2025) — target_sources、FILE_SET、C++20 模块
- **CMake: C'mon, it's 2025 already!** — Raúl Huertas (Using std::cpp 2025) — 2025 年的 CMake 最佳实践
- **CMake for the Impatient** — Steve Love (Meeting C++ 2025) — 快速上手指南
- **Oh No! More Modern CMake** — Meeting C++ 2018 — CMake 3.12+ 的进阶用法

## 社区与工具

- [CMake GitLab 仓库](https://gitlab.kitware.com/cmake/cmake) — 源码和 Issue 追踪
- [Kitware CMake 培训课程](https://www.kitware.com/courses/) — 官方付费培训
- [NERSC CMake Best Practices (Feb 2026)](https://www.nersc.gov/users/training/past-training/cmake-feb2026/) — HPC 场景最佳实践

## 博客文章

- [It's Time to Do CMake Right](https://pabloariasal.github.io/2018/02/19/its-time-to-do-cmake-right/) — Modern CMake 实践指南
- [Embracing Modern CMake](https://steveire.wordpress.com/) — Modern CMake 理念阐述

- [google/googletest](https://github.com/google/googletest) — FetchContent 集成测试框架的标准参考
