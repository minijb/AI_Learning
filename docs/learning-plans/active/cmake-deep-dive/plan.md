# 学习计划: CMake 深度剖析

> 创建日期: 2026-06-10
> 预计总耗时: 35-45 小时
> 目标水平: 精通

---

## 学习目标

完成本计划后，你能做到：

1. 从零搭建一个多目标、多依赖的 C/C++ 项目构建系统
2. 理解 CMake 内部架构——配置阶段、生成阶段、构建阶段各发生什么
3. 熟练使用生成器表达式实现条件编译、跨平台构建
4. 编写可复用的 `find_package` 模块和 `FetchContent` 依赖管理
5. 配置工具链文件实现交叉编译（ARM、Emscripten、iOS 等）
6. 使用 `CMakePresets.json` 标准化团队构建配置
7. 掌握 `install()`/`export()` 构建可分发的 CMake 包
8. 用 CTest + CPack 实现测试和打包流程
9. 调试 CMake 构建问题，理解缓存机制和作用域规则
10. 了解 CMake 4.0 的演进方向和 Post-Modern CMake 理念

## 前置要求

- [x] C/C++ 基础编程能力
- [x] 命令行操作基础
- [ ] 无需 CMake 先验知识（从零开始）
