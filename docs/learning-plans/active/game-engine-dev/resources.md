# 学习资源: 游戏引擎开发工程师

## 官方文档

- [OpenGL 官方文档](https://www.opengl.org/documentation/) — OpenGL API 参考
- [Vulkan 官方文档](https://www.vulkan.org/learn) — 现代 GPU API 规范
- [DirectX 12 文档](https://learn.microsoft.com/en-us/windows/win32/directx12/directx-12-programming-guide) — 微软图形 API
- [glTF 规范](https://registry.khronos.org/glTF/) — 3D 资产传输格式
- [CMake 文档](https://cmake.org/documentation/) — 跨平台构建系统

## 推荐书籍

### 图形学
- **《Real-Time Rendering, 4th Edition》** — Tomas Akenine-Möller 等 — 实时渲染圣经
- **《Physically Based Rendering, 3rd Edition》** — Matt Pharr 等 — PBR 理论和实践
- **《Fundamentals of Computer Graphics, 5th Edition》** — Steve Marschner 等 — 计算机图形学基础
- **《Computer Graphics: Principles and Practice, 3rd Edition》** — John F. Hughes 等 — 图形学经典教材

### 游戏引擎
- **《Game Engine Architecture, 3rd Edition》** — Jason Gregory — 游戏引擎架构权威指南
- **《Mathematics for 3D Game Programming and Computer Graphics》** — Eric Lengyel — 游戏数学
- **《Game Physics Engine Development》** — Ian Millington — 物理引擎开发
- **《Real-Time Collision Detection》** — Christer Ericson — 碰撞检测经典
- **《3D Math Primer for Graphics and Game Development》** — Fletcher Dunn 等 — 3D 数学入门

### C++ 与性能
- **《Effective Modern C++》** — Scott Meyers — 现代 C++ 11/14 核心特性与最佳实践
- **《C++ Concurrency in Action, 2nd Edition》** — Anthony Williams — C++ 并发与内存模型
- **《Optimized C++》** — Kurt Guntheroth — C++ 性能优化
- **《C++ Templates: The Complete Guide, 2nd Edition》** — David Vandevoorde — 模板元编程权威参考
- **《What Every Programmer Should Know About Memory》** — Ulrich Drepper — 内存系统

## 开源项目

### 完整引擎
- **[Godot Engine](https://github.com/godotengine/godot)** — 开源 2D/3D 游戏引擎，C++
- **[Bevy](https://github.com/bevyengine/bevy)** — Rust 编写的数据导向游戏引擎，ECS 架构
- **[Filament](https://github.com/google/filament)** — Google 的 PBR 渲染引擎
- **[bgfx](https://github.com/bkaradzic/bgfx)** — 跨平台图形抽象层

### 物理引擎
- **[Bullet Physics](https://github.com/bulletphysics/bullet3)** — 开源物理引擎
- **[Box2D](https://github.com/erincatto/box2d)** — 2D 物理引擎
- **[Jolt Physics](https://github.com/jrouwe/JoltPhysics)** — 现代 C++ 物理引擎

### 相关工具库
- **[GLFW](https://github.com/glfw/glfw)** — 跨平台窗口和输入
- **[glm](https://github.com/g-truc/glm)** — OpenGL 数学库
- **[stb](https://github.com/nothings/stb)** — 单头文件 C 库集合（图像、音频、字体等）
- **[assimp](https://github.com/assimp/assimp)** — 3D 模型导入库
- **[miniaudio](https://github.com/mackron/miniaudio)** — 单头文件音频库
- **[spdlog](https://github.com/gabime/spdlog)** — 快速 C++ 日志库
- **[Dear ImGui](https://github.com/ocornut/imgui)** — 即时模式 GUI 库

### 开源引擎（学习用）
- **[Godot Engine](https://github.com/godotengine/godot)** — 开源 2D/3D 游戏引擎，节点系统架构
- **[Filament](https://github.com/google/filament)** — Google 的 PBR 渲染引擎，FrameGraph 架构
- **[bgfx](https://github.com/bkaradzic/bgfx)** — 跨平台图形抽象层，命令缓冲模式
- **[Jolt Physics](https://github.com/jrouwe/JoltPhysics)** — 现代 C++ 物理引擎
- **[EnTT](https://github.com/skypjack/entt)** — 高性能 ECS 框架，稀疏集实现
