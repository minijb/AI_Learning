# 学习计划: C++ 内存模型

> 创建日期: 2026-06-08
> 预计总耗时: 8–10 小时
> 目标水平: 进阶

---

## 学习目标

完成本计划后，你能够：
- 预测任意 `struct` / `class` 的内存布局（大小、对齐、padding）
- 理解虚函数表、继承、虚继承在内存中的实际形态
- 正确使用 `new/delete`、`placement new`、对齐分配
- 安全地使用 `memcpy`/`memmove`/`memset` 等底层内存操作
- 理解 C++11 内存模型的基础概念：sequenced-before、happens-before、原子操作
- 在引擎级别代码中诊断内存布局相关性能问题和 UB

## 前置要求

- [x] 熟悉 C/C++ 基础语法（指针、引用、函数、类）
- [x] 了解操作系统虚拟内存的基本概念（堆/栈）
- [x] 有一台可编译 C++17 及以上标准代码的环境

## 学习路径

| 序号 | 知识点 | 预计耗时 | 类型 | 前置 |
|------|--------|---------|------|------|
| 1 | 字节、字、地址与整数内存表示 | 45min | 基础 | 无 |
| 2 | `struct` 与数组的内存布局 | 50min | 基础 | 1 |
| 3 | 类对象的内存模型 | 55min | 基础 | 2 |
| 4 | 继承、虚函数与虚继承 | 60min | 核心 | 3 |
| 5 | `new`/`delete` 与内存生命周期 | 50min | 核心 | 3 |
| 6 | Placement New 与对齐存储 | 50min | 进阶 | 5 |
| 7 | 底层内存操作（`memcpy` 等） | 45min | 进阶 | 2 |
| 8 | C++ 内存模型与原子操作 | 60min | 进阶 | 7 |

## 里程碑

- [ ] **第一阶段 (第 1–3 节)**：你能手绘任意 class 的内存布局图，预测 `sizeof` 和 `alignof`。
- [ ] **第二阶段 (第 4–6 节)**：你能安全地使用内存分配原语，理解 `new` 和 `placement new` 的区别，以及何时需要手动管理对齐。
- [ ] **第三阶段 (第 7–8 节)**：你能正确使用底层内存操作并避免 UB，理解 C++11 内存模型的核心概念（happens-before）。
- [ ] **最终项目**：从零实现一个类型安全的 Bump Allocator，支持任意对齐、原地构造、批量析构。
