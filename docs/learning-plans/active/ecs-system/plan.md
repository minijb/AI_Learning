# 学习计划: ECS 系统 — 从原理到实践

> 创建日期: 2026-05-31
> 预计总耗时: 约 60 小时（含练习）
> 目标水平: 精通

---

## 学习目标

完成本计划后，你将能够：

- 深刻理解 ECS 架构的核心思想（Entity、Component、System），以及它与 OOP、数据导向设计的关系
- 解释 Archetype/Chunk 存储模型如何实现 CPU 缓存友好和高效迭代
- 独立实现一个最小化但功能完备的 ECS 框架（C++ 或 Rust）
- 在 Unity DOTS 生态中构建高性能游戏逻辑
- 在 Unreal Engine 中使用 Mass 框架实现大规模实体模拟
- 评估并选用合适的开源 ECS 库（EnTT、Flecs、Bevy ECS 等）到实际项目
- 判断 ECS 模式是否适合特定问题域，避免反模式

## 前置要求

- [x] C++ 基础：熟悉 C++11/14 以上特性（模板、lambda、智能指针）
- [x] 理解 CPU 缓存层次结构和局部性原理的基本概念
- [ ] 了解 Unity 或 Unreal Engine 基础用法（可选，对第三、四部分有帮助）
- [ ] 了解 Rust 基础语法（可选，仅第五部分 Bevy/Legion 需要）

- [ ] **第一阶段: ECS 基础打通** — 完成第 1-5 节，能解释 Entity/Component/System 各自职责，能写出简单的伪 ECS 代码
